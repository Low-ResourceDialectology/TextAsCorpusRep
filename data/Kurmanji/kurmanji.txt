# Kurdish (Kurmanji) Openly Available Datasets
# %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

- 2023AhmadiSouthernCorpus/ → [[Ahmadi et al., 2023 - Southern Kurdish and Laki]]
Name: Language Identification of Kurdish & Zaza-Gorani Languages
Authors: Ahmadi, Sina and Azin, Zahra and Belelli, Sara and Anastasopoulos, Antonios
Description: Approaches to Corpus Creation for Low-Resource Language Technology: the Case of Southern Kurdish and Laki
URL: 
Github: https://github.com/sinaahmadi/KurdishLID/tree/main
Content: 2400 lines of Kurmanji text in latin script and also in arabic script
Size: 
Note: Also contains data for Central Kurdish, Southern Kurdish, Gorani, Zazaki, Arabic, Persian and Turkish
Access: git clone git@github.com:sinaahmadi/KurdishLID.git

- 2022AhmadiInterdialect/ → [[Ahmadi et al., 2022 - Kurdish Parallel Corpus]]
Name: Kurdish Parallel Corpus
Authors: Sina Ahmadi and Hossein Hassani and Daban Q. Jaff
Description: A parallel corpus of Kurdish (Sorani and Kurmanji) and English
URL: 
Github: https://github.com/KurdishBLARK/InterdialectCorpus
Content: 12,327 Sorani-Kurmanji, 1,797 Kurmanji-English, 650 Sorani-English translation pairs
Size: 34,90134,207 Kurmanji-English, 11,22011,855 Sorani-English, 205,382231,559 Sorani-Kurmanji Tokens
Note:
Access: git clone git@github.com:KurdishBLARK/InterdialectCorpus.git

- 2020AhmadiKurdishTokenization/ → [[Ahmadi, 2020 - Kurdish Tokenization]]
Name: Kurdish Tokenization 
Authors: Sina Ahmadi
Description: A Tokenization System for the Kurdish Language (Sorani & Kurmanji dialects)
URL: 
Github: https://github.com/sinaahmadi/KurdishTokenization
Content: 9,970 Headwords and 1,507 lemmata for Kurmanji, 8,180 Headwords and 1,513 lemmata for Sorani
Size: 
Note: 
Access: git clone git@github.com:sinaahmadi/KurdishTokenization.git

- 2013EsmailiPewan/ → [[Esmaili et al., 2013]] (Esmaili and Salavati published multiple works in the same year with very similar content)
Name: pewan
Authors: Kyumars Sheykh Esmaili, Shahin Salavati
Description: 
URL: 
Github: https://github.com/klpp/pewan/
Content: Northern Kurdish and Central Kurdish
Size: 25,572 new articles = 127,272 words
Note: 
Access: git clone https://github.com/klpp/pewan/ → extract the **Pewan.zip** → then extract the **Kurmanji.zip**

- 2020AhmadiKurdishMachineTranslation → [[Ahmadi and Masoud, 2020 - Towards Kurdish MT]]
Name: Kurdish Machine Translation
Authors: Sina Ahmadi, Mariam Masoud
Description: Towards Machine Translation for the Kurdish Language
URL: https://arxiv.org/abs/2010.06041
Github: https://github.com/sinaahmadi/KurdishMT#kurdish-machine-translation
Content: 
Size: 
Note: This work used multiple datasets such as Tanzil (Tiedemann, 2012), TED (Cettolo et al., 2012), KurdNet (Aliabadi et al., 2014), Pewan (Esmaili et al., 2013), Kurdish Textbooks Corpus (Abdulrahman et al., 2019), Veisi (Veisi et al., 2020), Folkloric (Ahmadi et al., 2020a)
Access: git clone git@github.com:sinaahmadi/KurdishMT.git
