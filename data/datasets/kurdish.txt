# Kurdish (Kurmanji) Openly Available Datasets
# %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

InterdialectCorpus → [[Ahmadi et al., 2022 - Kurdish Parallel Corpus]]
→ Leveraging Multilingual News Websites for Building a Kurdish Parallel Corpus
→ GitHub: https://github.com/KurdishBLARK/InterdialectCorpus
→ Data
	→ 34,90134,207 (Kurmanji English) 11,22011,855 (SoraniEnglish) 205,382231,559 (Sorani Kurmanji) Tokens
	→ corpus contains (12,327 SoraniKurmanji) (1,797 KurmanjiEnglish) (650 SoraniEnglish) translation pairs
	→ News articles from websites

KurdishTokenization → [[Ahmadi, 2020 - Kurdish Tokenization]]
→ A Tokenization System for the Kurdish Language
→ GitHub: https://github.com/sinaahmadi/KurdishTokenization
→ Data
	→ (9,970 Headwords for Kurmanji, and 1,507 lemmata) (8,180 Headwords, and 1,513 lemmata for Sorani)
	→ 100 sentences
	→ Used corpus FreeDicts, the Kurdish Wiktionary, Wîkîferheng. and lexicon Wergor of Sorani

pewan → [[Esmaili et al., 2013]] x2
→Building a Test Collection for Sorani Kurdish & Sorani Kurdish versus Kurmanji Kurdish: An empirical comparison
→ GitHub: https://github.com/klpp/pewan/
→ Data
	→Sorani: 115340 news articles → 501054 words
	→ Kurmanji: 25572 new articles → 127272 words

KurdishMT → [[Ahmadi and Masoud, 2020 - Towards Kurdish MT]]
→ Towards Machine Translation for the Kurdish Language
→ GitHub: https://github.com/sinaahmadi/KurdishMT
→ Data
	→ **Parallel Corpora**
		→ Tanzil corpus (Tiedemann, 2012) - 92,354 parallel sentences from the [Tanzil Project](http://tanzil.net)
		→ [TED corpus](https://wit3.fbk.eu) (Cettolo et al., 2012) 
		→ KurdNet (Aliabadi et al., 2014) - the Kurdish wordnet (parallel corpus?)
	→ **Raw Corpora**
		→ Pewan corpus containing 18M words (Esmaili et al., 2013)
		→ the Kurdish Textbooks Corpus (KTC) containing 693,800 words (Abdulrahman et al., 2019)
		→ Veisi et al’s corpus containing 8.1M words (Veisi et al., 2020) 
		→ Sorani Kurdish folkloric lyrics corpus containing 49,582 words (Ahmadi et al., 2020a)
